# SIPENA Backend

Skeleton backend ini disiapkan untuk migrasi dari Supabase langsung ke API custom.

Endpoint target awal:
- `GET /api/health`
- `POST /api/auth/login`
- `POST /api/auth/register`
- `POST /api/auth/logout`
- `GET /api/auth/me`
- `GET /api/users/me`
- `PATCH /api/users/me`
- `POST /api/files/upload`
- `GET /api/notifications`
